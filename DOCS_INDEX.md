## 🎯 开发者工作流

### 新开发者加入
1. 阅读 **README.md** - 了解开源项目
2. 阅读 **PRD.md** - 理解产品定位 (开源 + Pro License)
3. 阅读 **Help.md** - 搭建开发环境
4. 查看 **ROADMAP.md** - 了解当前进度
5. 开始贡献!

### 日常开发
1. 查看 **ROADMAP.md** - 确认当前任务
2. 参考 **PRD.md** - 确保符合产品定位
3. 更新 **CHANGELOG.md** - 记录变更
4. 更新 **ROADMAP.md** - 标记完成状态

## ⚠️ 重要提醒

### 产品定位核心原则
```
ComputeHub = 开源管理平台，不是 GPU 提供商
商业模式 = 开源 (MIT) + Pro License ($49 终身)
核心价值 = 自动化调度 + 统一管理 + 社区驱动
```

### 开发决策参考顺序
1. **PRD.md** - 产品定位和功能规划 (开源 + Pro)
2. **ROADMAP.md** - 当前优先级 (License 系统优先)
3. **Help.md** - 技术实现细节

### Free vs Pro 功能划分
- **Free (开源)**: 核心管理功能，价格对比，部署管理
- **Pro ($49 终身)**: 自动化、通知、高级监控、批量操作
