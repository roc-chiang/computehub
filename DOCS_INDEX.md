## 📌 核心文档 (必读)

### 1. **PRD.md** - 产品需求文档 ⭐⭐⭐
**用途**: 产品定位、商业模式、功能规划  
**受众**: 所有开发者、产品经理  
**重要性**: 🔥🔥🔥 最高优先级

**核心内容**:
- 产品定位: GPU 聚合平台 + 自动化调度
- 商业模式: **纯订阅制 SaaS** (不按 GPU 使用量计费)
- 订阅方案: Free ($0) / Pro ($49/月) / Team ($299/月)
- 核心价值: 统一管理 + 自动调度 + 跨云编排

**关键理念**:
> ComputeHub 不是 GPU 提供商,是管理平台  
> 用户付费给 ComputeHub 是为了管理和自动化服务  
> GPU 成本由用户直接付给 RunPod/Vast 等供应商

## 🎯 开发者工作流

### 新开发者加入
1. 阅读 **PRD.md** - 理解产品定位
2. 阅读 **Help.md** - 搭建开发环境
3. 查看 **ROADMAP.md** - 了解当前进度
4. 开始开发!

### 日常开发
1. 查看 **ROADMAP.md** - 确认当前任务
2. 参考 **PRD.md** - 确保符合产品定位
3. 更新 **CHANGELOG.md** - 记录变更
4. 更新 **ROADMAP.md** - 标记完成状态
